XPath Tool

Extract, edit, and evaluate XPath queries with ease.

XPath Tool makes it easy to extract, edit, and evaluate XPath queries on any webpage.

IMPORTANT: After installing this extension, reload any existing tabs or restart Chrome for the extension to work properly.

Instructions:

  1. Open a new tab and navigate to any webpage.
  2. Press Ctrl + Shift + X (or Command + Shift + X on macOS).
  3. Hold down Shift and move your mouse over elements. The query box updates with the XPath, and the results box shows matches.
  4. Edit the XPath query if needed; results update in real time.
  5. Repeat step 2 to close the console.
